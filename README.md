drone.io notifier
==============

An extension for drone.io build notification.

See https://chrome.google.com/webstore/detail/droneio-notifier/obggmnfdbpobnabjjfpakablffjkpheh

## Development

1. Setup
  ```sh
npm install
npm install -g bower
bower install
  ```

2. Build
  ```sh
grunt build
# => dist/*
  ```

3. Packaging
  ```sh
grunt pack
# => package/drone-notifier.zip
  ```
